# Использование

Node.js v16.16.0
[Ссылка на nvm windows (переключать версии node)](https://docs.microsoft.com/ru-ru/windows/dev-environment/javascript/nodejs-on-windows)

Устанавливаем

```
npm install
```

Разработка

```
npm run dev
```

Сборка для продакшена

```
npm run build
```

Проверка сборки для продакшена

```
npm run preview
```
